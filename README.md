## Features
- [x] Cross-platform / Easy to deploy
- [x] Multiple mode, Multiple storages
- [x] Multiple theme
- [x] Dark mode
- [x] File preview (image, video, audio, code, pdf, epub, markdown)
- [x] Download direct link
- [x] Anti-leech
- [x] Short link / Share link
- [x] Access control
- [x] Bypass
- [x] WebDav

## Supported Storage
- Local directory
